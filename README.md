# Spanish Tenses Web Crawler

## Description
This Python program is designed for aiding Latin language learners to memorize the verb conjugations.

Included Spanish tenses:
* Presente
* Futuro
* Pretérito imperfecto
* Pretérito perfecto compuesto
* Pretérito pluscuamperfecto
* Pretérito anterior
* Futuro perfecto

So far the list contains 783 verbs: https://rb.gy/oodbak

## Configuration
Preferred IDE: PyCharm

How to run the program:
1. Open Verb List.csv and add target verbs
2. Execute main.py
3. Result will be output in Microsoft Excel format as "Tiempos En Español"

## Utility
The result can be used in Text-to-Speech applications like: https://ttsreader.com/
